## Get started

Upon your first login to MocklyAPI, you'll be onboarded with a new organization and project automatically created.
To create additional organizations, use the organization switcher and select the option to create a new one.

<Frame>
  <img src="/images/create-organization.png" alt="Screenshot of organization switcher." style={{ borderRadius: '0.5rem' }} />
</Frame>

<Frame style={{ marginTop:'1rem' }}>
  <img src="/images/create-organization-modal.png" alt="Screenshot of organization switcher." style={{ margin:'10px', borderRadius: '0.5rem' }} />
</Frame>

<Callout icon="dollar-sign" color="#407C8F" iconType="regular">You will be billed for organization basis</Callout>

Create a new mock API by selecting a project and providing a name. Optionally, add a description.

<Frame>
  <img src="/images/create-mock-api-modal.png" alt="Screenshot of mock api create modal." style={{ borderRadius: '0.5rem' }} />
</Frame>

You'll be redirected to the editor view.

<Frame>
  <img src="/images/mock-api-editor-page.png" alt="Screenshot of mock api editor page." style={{ borderRadius: '0.5rem' }} />
</Frame>

Generate data by chatting with the system. Provide details for your dataset, e.g., "Create 10 users for an ecommerce clothing store application."

<Frame>
  <img src="/images/mock-api-editor-with-data-page.png" alt="Screenshot of mock api editor page." style={{ borderRadius: '0.5rem' }} />
</Frame>


Refine the dataset via additional chats if needed. Once satisfied, click "Save Endpoint" to persist changes.
Access the data by visiting the provided URL or sending an HTTP request to the endpoint.

<Danger>By default, endpoints are private, requiring an auth token via query parameters to prevent accidental exposure. Make them public via settings if desired.</Danger>

<Frame>
  <img src="/images/mock-api-editor-url.png" alt="Screenshot of mock api editor page." style={{ borderRadius: '0.5rem' }} />
</Frame>

Visit the endpoint to retrieve results.

<Frame>
  <img src="/images/api-json-result.png" alt="Screenshot of mock api editor page." style={{ borderRadius: '0.5rem' }} />
</Frame>
