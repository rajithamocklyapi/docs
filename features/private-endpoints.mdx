---
title: 'Private endpoints'
description: 'MocklyAPI gives authorized access to your mock apis'
icon: 'lock'
---

MocklyAPI provide private endpoints to make your mock api endpoints secure.

Once private endpoint is enabled to access your mock api you need to pass `auth_token` as query parameter to your endpoint.

<Note>This option is enabled by default to protect you mock api endpoints.</Note>

<Frame>
  <img src="/images/private-endpoint-token.png" alt="private endpoint settings token view." style={{ borderRadius: '0.5rem' }} />
</Frame>

You can always disable this feature to make your mock api endpoint public.

<Frame>
  <img src="/images/private-endpoint-settings.png" alt="private endpoint settings modal." style={{ borderRadius: '0.5rem' }} />
</Frame>

<Warning>Make sure to save endpoint to make change settings take effect</Warning>
